# 오케스트레이션 명령어

여러 AI CLI를 조합하여 복합 작업을 수행합니다.

## 사용 예시
- `/orchestrate "codex로 코드 생성 → gemini로 리뷰"`
- `/orchestrate "gemini로 분석 → codex로 수정"`

## 워크플로우
1. 작업 파이프라인 파싱
2. 각 단계별 서브 에이전트 호출
3. 중간 결과 전달 및 최종 결과 집계

## 파이프라인 패턴

### 패턴 1: 생성 → 리뷰
```
codex (코드 생성) → gemini (코드 리뷰) → 결과 반영
```

### 패턴 2: 분석 → 수정
```
gemini (문제 분석) → codex (코드 수정) → gemini (검증)
```

### 패턴 3: 병렬 실행
```
[codex (옵션 A)] + [codex (옵션 B)] → 비교 후 선택
```
