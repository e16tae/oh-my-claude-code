# 상태 확인 명령어

서브 에이전트의 상태와 설정을 확인합니다.

## 출력 정보
- 활성화된 서브 에이전트 목록
- 각 에이전트의 설정 프로필
- 최근 호출 이력 및 성공률

## 사용법
```
/status
```

## 출력 예시
```
📊 서브 에이전트 상태

| 에이전트 | 상태 | 모델 | 최근 호출 |
|---------|------|------|----------|
| codex-cli | ✅ 활성 | o4-mini | 5분 전 |
| gemini-cli | ✅ 활성 | gemini-2.5-pro | 10분 전 |
```
